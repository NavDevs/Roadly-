How It Works\n\n1. **Login**: Enter your mobile number to get started\n2. **Explore**: View nearby road incidents on the home screen\n3. **Report**: Tap the red button to report a new incident\n4. **Select**: Choose incident type (Accident, Road Work, Congestion, Blocked)\n5. **Details**: Add description and optional photo\n6. **Submit**: Report goes live instantly\n7. **Earn**: Collect points and climb the leaderboard\n\n## ??
